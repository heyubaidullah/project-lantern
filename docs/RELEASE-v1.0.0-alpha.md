# Release Notes — v1.0.0-alpha

## Summary
This alpha release establishes the core Al-Huda experience: a calm, guided daily journey that helps users read, understand, reflect, and act on one passage at a time.

## Included in This Release

### Core Flow
- Home page
- Onboarding flow
- Journey page
- Progress page
- About page

### Content Integration
- live chapter data
- live verse fetch
- live translation fetch

### User Activity
- save reflection
- save chosen action step
- progress/history rendering

### App Shell
- polished header
- navigation links
- clickable chapter preview cards
- footer
- theme toggle
- light / dark / system support

## Known Limitations
- no authentication yet
- no real user profile yet
- no streak tracking yet
- no first-name personalization yet
- no internal chapter reader yet
- chapter preview links currently open Quran.com

## Why Auth Is Excluded from Alpha
Authentication is intentionally deferred so the alpha can validate the product’s core journey before adding more complexity.

## Next Planned Area
Next major implementation area:
- authentication
- profile
- streaks
- welcome flow

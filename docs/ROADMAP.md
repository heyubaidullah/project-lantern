# Roadmap

## v1.0.0-alpha
Goal: validate the core experience.

Included:
- guided onboarding
- daily journey flow
- pathway-based experience
- live content integration
- reflection saving
- progress/history
- app shell polish
- theme support
- about page

Excluded:
- authentication
- profile
- streaks
- social features

## v1.0.0-beta
Goal: establish a real authenticated and personalized user journey.

Included:
- Supabase Auth
- Google sign-in
- email magic link sign-in
- user-linked profile storage
- onboarding with first name and last name
- per-user persistence
- per-user journey progression
- streak tracking
- profile page
- settings page
- privacy page
- disclaimer page
- deployed production build
- successful QA

Deferred:
- editable profile details
- interactive settings controls
- auth-aware guided pathway CTA behavior on Home
- social/circle features

## v1.0.0
Goal: polish and formalize release quality.

Potential additions:
- stronger visual consistency
- final wording cleanup
- final responsive/mobile polish
- profile editing
- smarter signed-in dashboard behavior
- refined pathway CTA behavior
- release documentation cleanup

## v1.1.0
Goal: deepen personalization and habit-building.

Planned:
- editable profile information
- richer settings
- stronger first-name personalization across the app
- improved progress summaries
- deeper pathway progression and completion behavior

## v1.2.0
Goal: make the experience more connected and motivating.

Planned:
- circles / groups / companion features
- shared reminders or accountability
- collaborative and social consistency features

## v2.0.0+
Goal: expand the product into a broader platform experience.

Planned:
- more flexible pathway authoring
- richer resource discovery
- advanced personalization
- native mobile applications

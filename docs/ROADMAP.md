# Roadmap

## v1.0.0-alpha
Goal: validate the core experience.

Included:
- guided onboarding
- daily journey flow
- pathway-based experience
- live content integration
- reflection saving
- progress/history
- app shell polish
- theme support
- about page

Excluded:
- authentication
- profile
- streaks
- social features

## v1.0.0
Goal: polish and formalize release quality.

Team expansion:
- Find & collaborate with more SMEs

Potential additions:
- stronger visual consistency
- final content and spacing polish
- refined motif/logo treatment
- better landing and welcome experience
- documentation cleanup

## v1.1.0
Goal: introduce real user identity.

Planned:
- Supabase Auth
- Google sign-in first
- email sign-in option
- user profile storage
- real per-user persistence
- user dropdown
- logout flow

## v1.2.0
Goal: deepen habit-building experience.

Planned:
- streak tracking
- streak display in navbar
- welcome-back behavior
- first-name personalization
- improved progress summaries

## v2.0.0+
Goal: expand beyond solo use.

Planned:
- groups/friends
- shared reminders or accountability
- collaborative pathways
- mobile applications
